# Changelog

## 2.0.0

### Major Update

This release introduces a complete redesign of the automation system.

Instead of the original toggle implementation, the mod now uses a more flexible action-based system, allowing for significantly more efficient automation with minimal player interaction.

### New automation modes

* Bow
* Fishing
* Swimming
* Sprint
* Cultivator / Hammer
* Eitr Magic

Additional improvements:

* Redesigned toggle logic
* Improved automation behavior
* Better overall usability and efficiency
